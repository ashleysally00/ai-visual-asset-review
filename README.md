# AI Visual Asset Review – Portfolio

This repository contains a focused portfolio of AI-generated 2D illustration assets created and curated for published coloring books.
The portfolio covers multiple themes and visual styles.

This portfolio highlights experience in:

- Prompt-driven visual asset creation using generative image models  
- Visual quality review and selection of AI-generated outputs  
- Style consistency and identity preservation across characters and scenes  
- Background composition and foreground separation for print-ready assets  

**Tools used:** Midjourney, DALL·E 3
